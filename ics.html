Industrial Control Systems Security Assessment

Cyber Physical Systems Security Project

Project Overview

This project focused on analysing Industrial Control Systems (ICS) and Operational Technology (OT) environments. The assessment involved discovering industrial assets, analysing communications and evaluating potential security risks.

Tools & Technologies

  • GrassMarlin
  • Nmap
  • Wireshark
  • Modbus Analysis
  • PLC Communications

Key Activities

  • Passive Asset Discovery
  • Network Enumeration
  • PLC Identification
  • Modbus Traffic Analysis
  • ICS Security Evaluation

Results

Successfully identified OT assets, mapped industrial communications and evaluated vulnerabilities within a cyber-physical environment.
