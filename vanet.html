Security Issues and Defence Mechanisms in VANETs

First Class Honours Dissertation Project

Project Overview

This dissertation investigated Black Hole attacks in Vehicular Ad Hoc Networks (VANETs) and proposed a hybrid Intrusion Detection System combined with a trust-based routing framework to improve network security and resilience.

Tools & Technologies

  • NS-2.35
  • Linux
  • Network Simulation
  • Intrusion Detection Systems
  • Trust-Based Routing

Key Objectives

  • Analyse Black Hole attacks in VANETs
  • Design a Hybrid IDS framework
  • Implement Trust-Based Routing
  • Evaluate network performance under attack

Results

The proposed solution improved packet delivery rates and enhanced the network's ability to identify malicious nodes while reducing the impact of Black Hole attacks.
