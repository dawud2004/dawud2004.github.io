Applied Penetration Testing Assessment

Team-Based Penetration Testing Engagement

Project Overview

This team project involved conducting a black-box penetration test against a target environment. The assessment followed industry-standard penetration testing methodologies and included exploitation, privilege escalation and reporting.

Tools & Technologies

  • Kali Linux
  • Nmap
  • Nikto
  • Nessus
  • FTP Client
  • Linux Exploitation Tools

Key Activities

  • Reconnaissance
  • Enumeration
  • Vulnerability Assessment
  • Exploitation
  • Privilege Escalation
  • Risk Assessment

Results

Successfully achieved user-level and root-level access, validated multiple vulnerabilities and provided remediation recommendations to improve the security posture of the target environment.
